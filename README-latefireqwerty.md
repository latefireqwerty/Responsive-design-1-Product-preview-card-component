### What I learned

Creating this responsive layout helped me practice relative units, media queries, and how to make a design work across different screen sizes. I also refreshed my understanding of flexbox and learned when it is best to use width-based sizing instead of fixed heights.

### Continued development

I want to keep practicing relative units, flexbox, grid, and responsive media queries. These concepts are my focus for the next project.
